### Install command line tools

Open your terminal, copy and paste the command below. Next, run it:

```bash
# linux/macOS
curl -o- -sL https://install.let-sh.com/install.sh | bash

# Windows
iwr https://install.let-sh.com/install.ps1 -useb | iex
```
